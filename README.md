# Javascript-POO
## Projeto elaborado durante um curso na Alura para aprender mais sobre programação orientada a objetos em Javascript

No curso de JavaScript: programando a Orientação a Objetos, eu tive a oportunidade de aprender sobre o paradigma da Programação Orientada a Objetos (POO). Aprendi sobre os conceitos fundamentais da POO, como referências e objetos, e como usar atributos, métodos da instância e da classe para criar objetos de maneira mais eficiente. Também aprendi sobre a criação de objetos através de construtores, o que me permitiu definir objetos de maneira mais precisa e flexível. Além disso, tive a oportunidade de aprender sobre o conceito de encapsulamento, o que me permitiu entender como proteger os atributos e métodos de um objeto de acessos indevidos. Enfim, o curso foi muito útil para aprimorar minhas habilidades em JavaScript e me tornar um programador mais eficiente.
